C++

¿Qué es C++?

C++ es una extensión de C que incluye programación orientada a objetos. Combina velocidad y control con capacidades modernas.

¿Cómo funciona C++?

Como C, se compila a código máquina, pero además permite organizar el código en clases y objetos. Es común en videojuegos, simulaciones, software financiero y más.
