SQL

¿Qué es SQL?

SQL (Structured Query Language) es un lenguaje diseñado para interactuar con bases de datos. Con SQL puedes guardar, buscar, actualizar y eliminar información de forma eficiente.

¿Cómo funciona SQL?

SQL se ejecuta en el servidor de la base de datos. Tú escribes consultas (queries) que son interpretadas por un sistema gestor como MySQL, PostgreSQL, SQL Server o SQLite. Se puede usar desde otras aplicaciones, como páginas web, para leer o modificar los datos almacenados.
