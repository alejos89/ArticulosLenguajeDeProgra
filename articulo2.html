CSS

¿Qué es CSS?

CSS (Cascading Style Sheets – Hojas de Estilo en Cascada) es el lenguaje que se usa para dar estilo y diseño a las páginas web. Con CSS puedes cambiar colores, fuentes, tamaños, posiciones y mucho más.

¿Cómo funciona CSS?

CSS trabaja junto con HTML. Mientras HTML estructura el contenido, CSS lo decora y organiza visualmente.
