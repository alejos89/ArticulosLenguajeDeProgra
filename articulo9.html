C#

¿Qué es C#?

C# (pronunciado "C sharp") es un lenguaje de programación moderno, orientado a objetos y desarrollado por Microsoft. Forma parte del ecosistema .NET y es muy usado para crear aplicaciones de Windows, sitios web, servicios en la nube, y videojuegos (especialmente con Unity).

¿Cómo funciona C#?

Cuando escribes un programa en C#, este se compila en un código intermedio llamado IL (Intermediate Language). Este código no se ejecuta directamente, sino que se interpreta por el CLR (Common Language Runtime), que forma parte de la plataforma .NET. Esto permite que el programa sea portable, seguro y optimizado en distintos sistemas.
