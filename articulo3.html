¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite hacer las páginas web interactivas. Con él puedes agregar cosas como botones que responden al hacer clic, menús dinámicos, validación de formularios, juegos, animaciones y mucho más.

¿Cómo funciona JavaScript?

JavaScript se ejecuta en el navegador del usuario, es decir, en el cliente. Puede trabajar junto con HTML y CSS para crear páginas web completas y dinámicas.
